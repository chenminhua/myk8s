### monitor k8s itself

```sh
helm install prometheus-kube prometheus-community/kube-prometheus-stack -n monitor

k port-forward svc/prometheus-kube-kube-prome-prometheus 9090 -n monitor
# access localhost:9090

sudo kubectl port-forward svc/prometheus-kube-grafana 80 -n monitor
# access localhost
# find admin password in values.yaml https://github.com/prometheus-community/helm-charts/blob/main/charts/kube-prometheus-stack/values.yaml#L513
```
